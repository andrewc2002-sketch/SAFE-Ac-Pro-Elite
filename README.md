# SAFE AC Pro Elite v4.3.1

## Upload to GitHub
1. Open the `SAFE-Ac-Pro-Elite` repository.
2. Tap **Add file** → **Upload files**.
3. Upload `index.html`, `manifest.json`, `service-worker.js`, and `icon.svg`.
4. Commit the files to the `main` branch.
5. Open **Settings** → **Pages**.
6. Choose **Deploy from a branch**, select `main`, select `/ (root)`, and save.

## Install on Android
Open the GitHub Pages site in Chrome, tap the three-dot menu, and choose **Install app** or **Add to Home screen**.
