# SAFE AC Pro Elite – Master

## Upload to GitHub
1. Extract this ZIP.
2. Upload the files inside the extracted folder to the root of the repository.
3. Replace existing files with the same names.
4. Commit to the `main` branch.
5. Wait for GitHub Pages to deploy.
6. Open the live site and press `Ctrl + Shift + R` once.
